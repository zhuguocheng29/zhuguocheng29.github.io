Hugo手工分类

2017-06-11 · 52 words · 1 minute read
2017年6月
hugo

Hugo手工分类弄了好长时间，根据自己找的主题+看文档，终于知道怎么手工分类了，Hugo默认分为两类，分别为categories和tags，用如下方法即可以自己分类：

    [taxonomies]
    tag = "tags"
    category = "categories"
    tool = "tools"

当然与此同时自己的菜单栏也需要进行相应的修改，对于相对于默认增加的分类，也需要自己手动添加相对应得html文件。

    [[params.menu]]
        before = true
        label  = "Home"
        link   = "/"

    [[params.menu]]
        before = false
        label  = "Categories"
        link   = "categories/"

    [[params.menu]]
        before = false
        label  = "Tags"
        link   = "tags/"

    [[params.menu]]
        before = false
        label  = "Tool"
        link   = "tools/"
